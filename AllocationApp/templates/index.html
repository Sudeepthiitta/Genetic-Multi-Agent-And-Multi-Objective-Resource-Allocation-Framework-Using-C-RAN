Project Overview

Cloud Radio Access Network separates BBUs and RRHs to improve scalability and centralized management. This project compares Fixed, Hybrid, and Multiobjective Genetic allocation strategies to reduce unfulfilled demand, increase resource utilization, and improve fairness across requests under dynamic traffic conditions.

How It Works

2) Allocation Runs

Execute Fixed, Hybrid, and Multiobjective algorithms to compare fulfillment behavior under shared capacity.

3) Metrics + Graph

Inspect output details, proof metrics, and graph comparison for optimization quality and load balance.
